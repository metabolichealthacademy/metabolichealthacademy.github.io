# Metabolic Health Academy

Evidence-based metabolic health education hub providing comprehensive resources on metabolic health, nutrition science, and lifestyle optimization.

## 📚 About

Metabolic Health Academy is an educational platform dedicated to sharing evidence-based information about metabolic health, including topics on nutrition, exercise, sleep, stress management, and disease prevention.

## 🌐 Website Structure

- **Home** (`index.html`) - Welcome page and main entry point
- **About** (`about.html`) - Information about the academy and mission
- **Resources** (`resources.html`) - Educational materials and references
- **Conditions** (`conditions.html`) - Information about metabolic health conditions
- **Blog** (`blog.html`) - Latest articles and updates
- **Contact** (`contact.html`) - Get in touch with us
- **Disclaimer** (`disclaimer.html`) - Medical and educational disclaimers
- **AI Disclaimer** (`ai-disclaimer.html`) - Information about AI-generated content
- **Privacy** (`privacy.html`) - Privacy policy
- **Terms** (`terms.html`) - Terms of service

## 🚀 Getting Started

This is a static HTML website hosted on GitHub Pages. To view the site, visit the domain configured in the `CNAME` file.

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/metabolichealthacademy/metabolichealthacademy.github.io.git
   cd metabolichealthacademy.github.io
   ```

2. Open any HTML file in your browser or use a local server:
   ```bash
   python -m http.server 8000
   # or with Node.js
   npx http-server
   ```

3. Navigate to `http://localhost:8000` in your browser
